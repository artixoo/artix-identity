# Utkarsh Pratham | Portfolio

A premium, tactical cybersecurity specialist portfolio built with React, Vite, and GSAP animations.

## Features

- **Silver Decryptor Theme**: Dark, tactical aesthetic with silver accents
- **Cinematic Preloader**: Name decryption animation with tactical sound effects  
- **Smooth Animations**: GSAP-powered transitions and reveals
- **Tactical FAQ Bot**: Browser-based intelligent assistant
- **Responsive Design**: Optimized for all devices

## Tech Stack

- React 19
- Vite 7
- GSAP animations
- TailwindCSS
- Lenis smooth scroll

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

The built files in `dist/` can be deployed to any static hosting service.
